# シェルスクリプト配布

https://github.com/101ta28/kit-proxy にてBash, Zshの両方に対応したシェルスクリプトを配布しています。

## インストール方法

```bash
$ curl -L "https://raw.githubusercontent.com/101ta28/kit-proxy/main/install.sh" | bash
```

## 使い方

```bash
# プロキシを有効にする
$ kit-proxy on

# プロキシを無効にする
$ kit-proxy off

# プロキシの設定を確認する
$ kit-proxy status

# スクリプトのヘルプを見る
$ kit-proxy help
```
